🎓 Campus Skill NFT

📌 Overview

Campus Skill NFT is a blockchain-based certification system that allows professors to issue non-transferable Skill NFTs to students as verifiable proof of their academic and technical achievements.

Unlike traditional certificates, these NFTs are stored on-chain, making them tamper-proof, publicly verifiable, and permanently linked to the student’s wallet address.

🚀 Problem Statement

Academic certificates and resumes can be forged or exaggerated.
There is no simple way for recruiters or institutions to verify whether a student genuinely possesses a claimed skill.

💡 Solution

Campus Skill NFT enables professors to mint Soulbound (non-transferable) NFTs containing:

- Student Name
- Skill Title
- Project / GitHub Link
- Issuer (Professor) Address
- Timestamp of issuance

Since the NFT is non-transferable, it acts as a permanent proof of achievement.

🔐 Key Features

✅ Only authorized professor can mint NFTs

✅ Non-transferable (Soulbound) token

✅ On-chain verification

✅ Publicly viewable skill record

✅ Deployed on Ethereum Sepolia Testnet

🛠 Tech Stack
